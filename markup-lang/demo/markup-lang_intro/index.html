Introducció a Llenguatge de Marques

Què és HTML?

HTML (HyperText Markup Language) és el llenguatge estàndard per estructurar i dissenyar pàgines web. Utilitza etiquetes per definir el contingut i no inclou lògica, sinó que descriu la semàntica i la disposició del contingut.

Principals característiques

Llenguatge de Marques
Utilitza etiquetes per estructurar diferents tipus de contingut, com paràgrafs, encapçalaments i taules.
Estructura jeràrquica
Organitza els elements en una estructura d'arbre (DOM), amb relacions entre pares i fills.
Semàntica
Inclou etiquetes com <header>, <section> i <footer> per descriure millor el contingut.
Independència de Plataforma
Funciona en qualsevol navegador modern, independentment del sistema operatiu.
Integració amb Altres Llenguatges
Es combina amb CSS per donar estil i amb JavaScript per afegir interactivitat.
Multimèdia
Permet integrar àudio, vídeo i imatges mitjançant etiquetes com <audio> i <video>.
Hipertext
Facilita la navegació entre pàgines mitjançant enllaços (<a>).
Fàcil d'Aprendre
La seva sintaxi senzilla el fa accessible per a principiants.
Extensible
Compatible amb APIs modernes com geolocalització o emmagatzematge local.
Estàndard en Evolució
Regularment actualitzat pel W3C i WHATWG. L'última versió és HTML5.

Què és CSS?

CSS (Cascading Style Sheets) és un llenguatge utilitzat per donar estil i format a les pàgines web. Treballa conjuntament amb HTML per controlar l'aparença visual del contingut, com colors, tipografies i disseny.

Principals característiques

Separació de Contingut i Disseny
Permet mantenir el contingut (HTML) separat del disseny (CSS), facilitant el manteniment del codi.
Fulls d'Estil en Cascada
Les regles s'apliquen segons la jerarquia i l'especificitat, combinant estils de diferents fonts.
Estilització de Tipografies
Permet personalitzar fonts, mides, colors i espaiats del text.
Disseny de Layouts
Ofereix eines com flexbox i grid per estructurar el contingut de manera eficient.
Responsivitat
Facilita l'adaptació de les pàgines a diferents dispositius mitjançant les media queries.
Compatibilitat amb Navegadors
Funciona en tots els navegadors moderns, amb algunes variacions en versions antigues.
Personalització Visual
Controla colors, marges, vores, fons i altres aspectes visuals de la pàgina.
Animacions i Transicions
Permet crear efectes visuals amb transicions, transformacions i animacions.
Modularitat
Permet reutilitzar estils mitjançant classes i fulls d'estil externs.
Interacció amb HTML i JavaScript
Treballa conjuntament amb HTML per estructurar el contingut i amb JavaScript per afegir interactivitat.

Què és JavaScript?

JavaScript és un llenguatge de programació d'alt nivell, interpretat i dinàmic, disenyat per afegir interactivitat a les pàgines web. És el llenguatge més utilitzat a l'entorn web per la part Front-End, i cada cop més utilitzat també al Back-End amb eines com Node-js.

Principals característiques

Llenguatge interpretat
No necessita ser compilat, els navegadors web tenen motors que interpreten i executen el codi directament.
Multiplataforma
És compatible amb la majoria dels navegadors sino tots.
Llenguatge dinàmic
No s'especifica explícitament el tipus de les variables, sino que es determina en temps d'execució.
Basat en prototips
No utilitza classes tradicionals, sino prototips per l'herencia i la creció d'objectes.
Orientat a events
Permet respondre a interaccions de l'usuari mitjançant "event listeners".
Llenguatge asíncrone
Soporta programació asíncrone a través de callbacks, promeses i "async/await".
Multiparadigma
Soporta diferents estil de programació, com programació funcional, programació orientada a objectes o programació imperativa.
Manipulació del DOM
Permet interactuar amb el DOM per modificar contingut, disseny o comportament de la web en temps real
